# AWS SAA Exam Notes

Concise study notes for the AWS Solutions Architect Associate exam, organized by service. Each folder includes a local `README.md` so you can browse topics from GitHub without opening files blindly.

## Where to Start

- Start with [IAM](./1.IAM/README.md) for identity, access control, and permissions basics.
- Continue with [EC2](./2.EC2/README.md) for compute concepts, instance choices, networking, and pricing models.

## Documentation Map

- [IAM](./1.IAM/README.md) - Notes on users, groups, policies, roles, credential reporting, and Access Advisor.
- [EC2](./2.EC2/README.md) - Notes on instance fundamentals, sizing, security groups, SSH access, IAM roles for EC2, and purchase options.

## Repository Layout

- [`1.IAM/`](./1.IAM/README.md) - Identity and access management notes.
- [`2.EC2/`](./2.EC2/README.md) - Elastic Compute Cloud notes.
